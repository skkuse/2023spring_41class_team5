# Backend

Node v16 required.

### Init Project

1. Get source code from github.

```
git clone https://github.com/skkuse/2023spring_41class_team5
cd 2023spring_41class_team5
```

2. Move to backend folder.

```
cd backend
```

3. Download node modules

```
npm install
```

4. Create .env file

```
PORT=
GPT_API_KEY=
...
```

### Start Server

```
npm run start
```
